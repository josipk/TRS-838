# TRS-838
TRS-838 Terminal

## Technical data:
Screen size: 12" (30 cm)
Technology: cathode ray tube
Colour: monochrome (green)
Number of columns: 80
Number of rows: 25
Mode: block, character
Video RAM: 6 kB, expandable to 12 kB (graphics display capability)
Character matrix: 7x9
Interfaces: 2 x RS-232, RS-422
Baud rate: 110-9600 bps
